# FrontEnd

### Structure
```sh
src
├── assets // static data
├── api // api hooks
├── components // components with styling
├── context // state and action contexts  
├── layout // base Layout sets
├── utils // utility functions
├── pages // routing pages: /(home), /about, etc...
├── styles // global/ constant styles
│   ├── globalStyles.js
│   └── constants.js
└── App.js // Root component
```
